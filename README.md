- Install

For install in Ubuntu run:

    go get gitlab.com/seriyyy95/morph
    go get github.com/conformal/gotk3

    sudo apt install libgtk-3-dev libcairo2-dev libglib2.0-dev

    git clone https://github.com/Seriyyy95/KeywordsMixer
    cd ./KeywordsMixer
    go build

Then run:

    ./KeywordsMixer
